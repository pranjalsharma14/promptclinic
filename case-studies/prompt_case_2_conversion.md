# Case Study 2 — Conversion Prompt Engineering

## Objective

Demonstrate how prompt structure can significantly improve conversion-focused AI content.

---

## Weak Prompt

Write a landing page for an online AI course.

---

## Diagnosis

The prompt lacks:

- target audience
- positioning
- persuasive elements
- emotional triggers
- call-to-action

Prompt Health Score: 7/50

---

## Optimized Prompt

Act as a high-conversion copywriter.

Write a landing page introduction for an AI Prompt Engineering course designed for freelancers and digital marketers who want to use AI tools professionally.

Use persuasive language, highlight the career advantage of prompt engineering skills, and create curiosity that motivates readers to explore the course.

Keep the tone confident, modern, and aspirational.

---

## Output Difference

Weak prompt output:

"This course teaches prompt engineering and how to use AI tools."

Optimized prompt output:

"AI is transforming how work gets done — but the real advantage belongs to those who know how to *communicate with AI effectively*. This Prompt Engineering course shows you how to turn AI into a powerful professional skill."

---

## Key Insight

Small improvements in prompt structure create large improvements in output persuasion and clarity.
